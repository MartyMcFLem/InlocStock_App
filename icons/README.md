# Icônes de l'application

Ce dossier doit contenir les icônes de l'application dans différentes tailles.

## Icônes nécessaires:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Génération des icônes

Vous pouvez créer ces icônes à partir d'une image source en utilisant:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Ou manuellement avec un outil comme GIMP, Photoshop, etc.

## Recommandations

- Utilisez un logo simple et reconnaissable
- Assurez-vous que l'icône est lisible même en petite taille
- Utilisez un fond de couleur unie (suggestion: #4F46E5 - indigo)
- Incluez une marge interne de 10% pour le mode "maskable"
